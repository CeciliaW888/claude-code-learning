Day 16: MCP 集成

🔴高级 · 预计时间: 1小时

MCP（Model Context Protocol）是 Anthropic 推出的开放协议，让 AI 模型能与外部工具和数据源无缝集成。Claude Code 原生支持 MCP，可动态发现和注册外部工具。

1
三种能力：Tools / Resources / Prompts

MCP 定义了三种能力类型：Tools（可执行的操作）、Resources（可读取的数据源）、Prompts（预定义的提示模板）。三者共同构成完整的扩展接口。

2
Server / Client 架构

MCP 采用 Server/Client 模式：Server 提供能力（如数据库查询工具），Client（Claude Code）发现并调用这些能力。通过 JSON-RPC 通信。

3
动态注册

MCP Server 启动后，Claude Code 自动发现其提供的工具并注册到工具列表中。无需重启，即可获得新能力。

关键洞察： MCP 让 Claude Code 的能力不再受限于内置工具——任何人都可以开发 MCP Server 来扩展其功能。

💻 核心代码

// MCP Server 示例
import { McpServer } from "@modelcontextprotocol/sdk/server/mcp.js";

const server = new McpServer({ name: "my-tools" });

// 注册工具
server.tool(
  "query_database",
  "查询数据库并返回结果",
  { sql: { type: "string", description: "SQL 查询语句" } },
  async ({ sql }) => {
    const result = await db.query(sql);
    return { content: [{ type: "text", text: JSON.stringify(result) }] };
  }
);

await server.connect(transport);
