Day 1: 架构总览

🟢入门 · 预计时间: 1小时

Claude Code 是一个基于终端的 AI 编程助手，理解其架构是深入学习的第一步。本节将带你从宏观视角认识整个系统的三层结构、Agent 循环机制以及统一的工具接口设计。

1
三层架构

Claude Code 采用经典的三层架构：表现层（终端 UI）、逻辑层（Agent 循环）和能力层（Tool 系统）。每层职责清晰，解耦良好。

2
Agent 循环：感知→思考→行动

核心循环不断重复三个步骤：感知用户输入和环境信息，思考下一步操作，执行工具调用。循环直到任务完成才停止。

3
30+ Tool 统一接口

所有工具（Bash、文件读写、搜索等）都遵循统一的 ToolDef 接口定义，包含名称、描述、输入 Schema 和执行函数，实现即插即用。

关键洞察： 三层架构让 Claude Code 既能保持简洁的用户界面，又能灵活扩展底层能力——理解这个分层是掌握整个系统的钥匙。

💻 核心代码

// Tool 统一接口定义
type ToolDef = {
  name: string;
  description: string;       // 给 Claude 看的描述
  inputSchema: JSONSchema;   // 参数校验
  call: (input: unknown) => Promise<ToolResult>;
};
