Day 4: REPL 与输入

🟡中级 · 预计时间: 1小时

REPL（Read-Eval-Print Loop）是 Claude Code 交互的核心引擎。本节深入解析输入处理、斜杠命令系统以及 Early Input 预捕获机制。

1
REPL.tsx 交互引擎

REPL 组件管理整个交互循环：等待用户输入、提交给 Agent 处理、显示结果、再次等待输入。这是用户与 AI 对话的桥梁。

2
斜杠命令系统

输入 /help、/clear、/model 等斜杠命令可以直接控制 Claude Code 的行为，无需经过 AI 处理，提供快捷操作入口。

3
Early Input 预捕获

在 Agent 处理期间，用户可以提前输入下一条消息。系统会缓存这些输入，等当前处理完成后立即提交，消除等待感。

关键洞察： Early Input 机制让用户无需等待 AI 响应完成就能继续输入，大幅提升交互流畅度。
