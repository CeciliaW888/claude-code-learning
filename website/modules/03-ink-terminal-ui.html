Day 3: 终端 UI (Ink)

🟢入门 · 预计时间: 1小时

Ink 将 React 的组件化开发模式带到了终端。Claude Code 使用 Ink 构建其终端界面，让 CLI 也能拥有丰富的交互体验。

1
React for CLI

Ink 是终端的 React 渲染器。你可以用 JSX 编写终端界面，使用 useState、useEffect 等 Hooks 管理状态和副作用。

2
Box = div，Text = span

Box 组件对应 HTML 的 div（容器布局），Text 组件对应 span（文本显示）。理解这个映射关系，就能快速上手 Ink。

3
Flexbox 布局

Ink 支持 Flexbox 布局模型，通过 flexDirection、justifyContent、alignItems 等属性控制终端中的元素排列。

关键洞察： 选择 Ink 而非传统 ncurses 方案，让 Claude Code 团队能用熟悉的 React 模式快速构建复杂终端 UI。

💻 核心代码

import { Box, Text } from "ink";

// 简单 Ink 组件示例
function StatusBar({ model, tokens }: Props) {
  return (
    <Box flexDirection="row" justifyContent="space-between">
      <Text color="green">模型: {model}</Text>
      <Text color="yellow">Token: {tokens}</Text>
    </Box>
  );
}
