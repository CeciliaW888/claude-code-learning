Day 8: BashTool

🟡中级 · 预计时间: 1小时

BashTool 是 Claude Code 最强大也最危险的工具。它允许 AI 在终端执行任意命令，因此需要多层安全机制来防止误操作。

1
AST 安全解析

使用抽象语法树（AST）解析 Bash 命令，而非简单的字符串匹配。这能准确识别 rm -rf、sudo 等危险操作，避免误报和漏报。

2
多层安全防护

安全检查分为多层：命令黑名单、AST 分析、用户确认、沙盒执行。每层独立工作，形成纵深防御。

3
输出截断

命令输出可能非常长（如 cat 大文件），系统会智能截断超长输出，保留头尾关键信息，避免消耗过多 token。

关键洞察： AST 解析是安全的核心——它能理解命令的真实意图，而不是被字符串伪装所欺骗。

💻 核心代码

// Bash 命令安全检查简化
function checkCommandSafety(command: string): Safety {
  const ast = parseBashAST(command);

  for (const node of ast.commands) {
    if (DANGEROUS_COMMANDS.has(node.name)) {
      return { safe: false, reason: `危险命令: ${node.name}` };
    }
    if (node.flags.includes("--force")) {
      return { safe: false, reason: "包含 --force 标志" };
    }
  }

  return { safe: true };
}
